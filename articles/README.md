## 📁 ディレクトリ構成

```
articles/
├── medical/     # 医療・薬剤関連の記事
├── it/          # IT・プログラミング関連の記事
├── misc/        # その他の雑記
└── TEMPLATE.md  # 記事テンプレート
```

## ✍️ 記事の書き方

1. `TEMPLATE.md` をコピーして新しい記事を作成
2. ファイル名は `YYYY-MM-DD-article-title.md` 形式で
3. 適切なカテゴリディレクトリに配置
4. 気軽に書く！完璧じゃなくてOK
